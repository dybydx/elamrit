This programme paired a structured seated course flow with one high-energy live station at the end so guests could choose their closing note — some wanted quiet coffee, others wanted the theatre of flambé and craft ice cream.

We built buffers around the awards block so nothing felt rushed when timings slipped, and kept dietary tables visually consistent with the main room so every guest felt included in the same occasion.
